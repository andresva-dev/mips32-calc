# MIPS32 Basic Calculator
 
A simple calculator written in MIPS32 assembly, developed and tested in **MARS** (MIPS Assembler and Runtime Simulator).
 
## Features
 
- Menu-driven interface
- Four basic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
## Requirements
 
- [MARS MIPS Simulator](http://courses.missouristate.edu/kenvollmar/mars/)
## Usage
 
1. Open the `.asm` file in MARS.
2. Assemble and run the program.
3. Select an operation from the menu.
4. Enter the two operands when prompted.
5. View the result in the console.
